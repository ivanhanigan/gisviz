GISViz

R tools to assist GIS visualisations.

Download this package as a .zip file Download this package as a tar.gz file

GISViz is an R package containing functions I've modified or created for visualising GIS data in population health and environmental science projects.

See the User Guide

Also see http://www.ada.edu.au/social-science/gisviz for related GISViz tools of particular use to social scientists.
